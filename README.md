# ian-legal

Legal documents for personal software projects owned and operated by Ian Collins.

This repository hosts the privacy policy and end user license agreement referenced
by small, personal, read-only data connectors and integrations (e.g. the
QuickBooks Online read-only MCP connector used with Claude Code).

These documents apply to software that:

- Is run locally on the owner's own machine
- Is used only by the owner, against the owner's own accounts
- Transmits data only between the owner's machine and the upstream service
  (e.g. QuickBooks Online) and performs no third-party sharing
